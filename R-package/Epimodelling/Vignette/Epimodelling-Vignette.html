1. What is Epimodelling?

Epimodelling is a tool to simulate and visualise data under different deterministic compartmental epidemiological models. It has two main functionality:

  1. Manual mode
  2. Interactive mode

In Manual mode you can choose model, solve the equations and visualise the results with few functions from R. In Interactive mode the user only calls the EpiSimulator() function which starts a Shiny web application.
